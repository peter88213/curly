yWriter.ott -- Document template for international customization

21 x 29,7 cm (A4) paper size
30 lines
60 characters/line
12 p Courier New
Document language: de-DE


HOW TO LOCALIZE YOUR OOTyW INSTALLATION

1. Copy "yWriter.ott" to your OpenOffice "..\user\template" directory. Overwrite existing file, if applicable.
2. Right-click "yWriter.ott" and select "open"  (DO NOT DOUBLE CLICK) --> This starts Office Writer to modify the template
3. (OpenOffice main menu) --> Tools --> Language --> For all Text 
4. Select your language
5. Save document.
